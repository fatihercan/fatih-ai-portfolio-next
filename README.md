# Fatih Ercan — Premium AI Portfolio

A premium Next.js + Tailwind portfolio positioning Fatih Ercan as an AI Language Specialist bridging language, education and artificial intelligence.

## Local preview

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy

Recommended: Vercel.

1. Upload this project to GitHub.
2. Go to Vercel.
3. Add New Project.
4. Import the GitHub repository.
5. Click Deploy.
